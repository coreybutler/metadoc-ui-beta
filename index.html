Metadoc

The online documentation platform where developers collaborate and learn from shared inline notes, tips and comments.

Documentation with Finesse

Metadoc delivers a powerful and intuitive documentation experience that blends seamlessly with your workflow.

Some of Metadoc's unique features include:

  • Bookmarks

    Save deep references to sections of documentation OR source code.

  • Notes

    Jot down your thoughts right in the documentation.

  • Custom Workspaces

    Keep all the docs that are relevant to your workflow together, in one place.

  • Scratch Pad

    Store sections of documentation along with Notes and Bookmarks for later reference.

Collaborate for Better Docs

Metadoc empowers developers to collaborate to improve documentation, boosting learning within teams and across the technology community.

  • Share Bookmarks and Notes with package authors and developers
  • Directly reference Github issues inline
  • Reference docs or source code from one package or library inside another

Starting from

Metadoc v1.0 will be focused on the Go Programming Language as a proof of concept. After that it will expand to support more languages, including JavaScript.
